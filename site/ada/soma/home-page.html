<HTML> element must have a valid lang attribute

Disabilities Affected:

  • Blindness
  • Deafblindness
  • Cognitive

Issue Type:

  • WCAG 2.0 (A): MUST

WCAG Success Criteria:

  • 3.1.1 Language of Page

Section 508 Guidelines:

Fix any of the following:

  • The <html> element does not have a lang attribute

Images must have alternate text

Disabilities Affected:

  • Blindness
  • Deafblindness

Issue Type:

  • WCAG 2.0 (A): MUST
  • Section 508: MUST

WCAG Success Criteria:

  • 1.1.1 Non-text Content

Section 508 Guidelines:

  • 1194.22 (a) Text equivalent for non-text elements
  • 1194.21 (a) Programmatically-discernable text

HTML:

Did not include 3rd party tracking pixels in this audit.

<img src="/store/ibf/images/flags/USA.jpg"> <img src="/web_assets/img/20160302/hp_poster.jpg"> <img src="/web_assets/footer/footer_social.gif" width="350" height="58" border="0" usemap="#socialMediafooterMap">

Fix any of the following:

  • Element does not have an alt attribute
  • aria-label attribute does not exist or is empty
  • aria-labelledby attribute does not exist, references elements that do not exist or references elements that are empty or not visible
  • Element has no title attribute or the title attribute is empty
  • Element's default semantics were not overridden with role="presentation"
  • Element's default semantics were not overridden with role="none"

<ul> and <ol> must only directly contain <li>, <script> or <template> elements

Lists must be marked up correctly, meaning they must contain at only <li elements and script supporting elements (<script> and <template>).

  • Blindness
  • Deafblindness

Issue Type:

  • WCAG 2.0 (A): MUST

WCAG Success Criteria:

  • 1.3.1 Info and Relationships

Section 508 Guidelines:

HTML:

<ul id="menu">

Fix any of the following:

  • List element has children that are not <li>, <script> or <template> elements

<video> elements must have a <track> for captions

Disabilities Affected:

  • Deafness

Issue Type:

  • WCAG 2.0 (AA): MUST
  • Section 508: MUST

WCAG Success Criteria:

  • 1.2.2 Captions (Prerecorded)

Section 508 Guidelines:

  • 1194.22 (a) Text equivalent for non-text elements
  • 1194.22 (b) Multimedia presentation alternatives synchronization

HTML:

<video tabindex="-1" class="video-stream html5-main-video" style="width: 994px; height: 559px; left: 0px; top: 0px; transform: none;" src="blob:https%3A//www.youtube.com/b701e3c0-45ea-4af6-9aad-d8ba7564bfea"></video>

Fix any of the following:

  • The multimedia element does not have a captions track
