Certain ARIA roles must contain particular children

Disabilities Affected:

  • Blindness
  • Deafblindness
  • Mobility/Dexterity

Issue Type:

  • WCAG 2.0 (A): MUST

WCAG Success Criteria:

  • 4.1.1 Parsing

HTML:

<div class="carousel-inner" role="listbox">

Fix any of the following:

Required ARIA child role not present: option

Buttons must have discernible text

Disabilities Affected:

  • Blindness
  • Deafblindness

Issue Type:

  • WCAG 2.0 (A): MUST
  • Section 508: MUST

WCAG Success Criteria:

  • 4.1.2 Name, Role, Value

Section 508 Guidelines:

  • 1194.22 (a) Text equivalent for non-text elements

HTML:

<a class="carousel-control" href="#carousel-collection-0" role="button" data-slide="next"></a>

Fix any of the following:

  • Element does not have inner text that is visible to screen readers
  • aria-label attribute does not exist or is empty
  • aria-labelledby attribute does not exist, references elements that do not exist or references
  • elements that are empty or not visible
  • Element's default semantics were not overridden with role="presentation"
  • Element's default semantics were not overridden with role="none"

Fix all of the following:

  • Element is in tab order and does not have accessible text

Elements must have sufficient color contrast

Disabilities Affected:

  • Low Vision
  • Color-blindness

Issue Type:

  • WCAG 2.0 (AA): MUST

WCAG Success Criteria:

  • 1.4.3 Contrast (Minimum)

Section 508 Guidelines:

  • 1194.21 (j) Color and contrast range

HTML:

<span itemprop="title">New Arrivals</span>

Fix any of the following:

Element has insufficient color contrast of 4.48 (foreground color: #777777, background color: #ffffff, font size: 9.0pt, font weight: normal)

HTML:

<div class="product-count" data-product-total="283"> 283 Items</div>

Fix any of the following:

Element has insufficient color contrast of 3.53(foreground color: #898888, background color: #ffffff, font size: 9.0pt, font weight: normal)

HTML:

<span>Size</span> <span>Ring Sizes</span> <span>Inseam</span> <span>Color</span> <span class="filter-option pull-left">Sort By</span> <span class="product-msg">Available in Regular and Petite</span>

Fix any of the following:

Element has insufficient color contrast of 3.70(foreground color: #7a7a7a, background color: #eeeeee, font size: 9.0pt, font weight: normal)
