Form elements must have labels

Disabilities Affected:

  • Blindness
  • Deafblindness
  • Low Vision
  • Mobility/Dexterity

Issue Type:

  • WCAG 2.0 (A): MUST
  • Section 508: MUST

WCAG Success Criteria:

  • 1.3.1 Info and Relationships
  • 3.3.2 Labels or Instructions
  • 4.1.2 Name, Role, Value

Section 508 Guidelines:

  • 1194.22 (n) Information required to complete electronic forms online
  • 1194.21 (l) Form fields, directions, and cues

HTML:

<input id="firstName" maxlength="20" name="firstName" value="" tabindex="1" class="single" type="text"> <input id="lastName" maxlength="40" name="lastName" value="" tabindex="2" class="single" type="text"> <input id="loyalityId" maxlength="20" name="loyalityId" value="" tabindex="3" class="single" type="text"> <select id="pCountry" name="pCountry" tabindex="3" class="single" onchange="displayCaslDiv()"> <input id="pAddress1" maxlength="40" name="pAddress1" value="" tabindex="4" class="single" type="text"> <input id="pAddress2" maxlength="40" name="pAddress2" value="" tabindex="5" class="single" type="text"> <input id="pCity" maxlength="30" name="pCity" value="" tabindex="6" class="single" type="text"> <input id="pState" maxlength="40" name="pState" value="" tabindex="7" class="single" type="text" size="10"> <input id="pPostalCode" maxlength="10" name="pPostalCode" value="" tabindex="8" class="single" type="text" size="10"> <input id="pPhoneNumber" maxlength="15" name="pPhoneNumber" value="" tabindex="10" class="single" type="text" size="15"> <input id="login" maxlength="60" name="login" onblur="if(this.value.replace(/^\s+|\s+$/g,'')==''){this.value='';document.getElementById('password').value='';document.getElementById('confirmPassword').value='';}" value="" tabindex="11" class="single" type="text" autocomplete="off"> <input id="password" maxlength="15" name="password" value="" tabindex="12" class="single" type="password"> <input id="confirmPassword" maxlength="15" name="confirmPassword" value="" tabindex="13" class="single" type="password"> <select id="birthMonth" name="birthMonth" tabindex="15" class="single"> <select id="referralSource" name="referralSource" tabindex="16" class="single">

Fix any of the following:

  • aria-label attribute does not exist or is empty
  • aria-labelledby attribute does not exist, references elements that do not exist or references elements that are empty or not visible
  • Form element does not have an implicit (wrapped) <label>
  • Form element does not have an explicit <label>
  • Element has no title attribute or the title attribute is empty
