Image buttons must have alternate text

Disabilities Affected:

  • Blindness
  • Deafblindness
  • Mobility/Dexterity

Issue Type:

  • WCAG 2.0 (A): MUST
  • Section 508: MUST

WCAG Success Criteria:

  • 1.1.1 Non-text Content

Section 508 Guidelines:

  • 1194.22 (a) Text equivalent for non-text elements

HTML:

<input name="/atg/userprofiling/ProfileFormHandler.forgotPassword" value="Reset Password" src="/store/images/buttons/btn_password_reset.gif" type="image">

Fix any of the following:

  • Element has no alt attribute or the alt attribute is empty
  • aria-label attribute does not exist or is empty
  • aria-labelledby attribute does not exist, references elements that do not exist or references elements that are empty or not visible
