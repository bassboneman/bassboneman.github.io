Atoms

Atoms are the smallest building blocks of the user interface. A label/input group, a single button, and an alert message are all examples of atoms.

Molecules

Molecules are groups of atoms that function as a single unit. For example, the atoms described above could be used to create an ecommerce checkout form, one that includes multiple fields, info messaging, and submit/cancel buttons.

Pages

Pages are made up of numerous atoms and molecules, including navigation, imagery, text blocks, web forms, and more.
